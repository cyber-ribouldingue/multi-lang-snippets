Snippets Multi-langages

Une collection d'extraits de code simples et multi-langages pour accélérer votre développement.

Langages supportés :

  • Python
  • JavaScript
  • HTML
  • CSS
  • Bash
  • Et plus encore...

Comment utiliser ?

Chaque répertoire contient des extraits de code pour un langage spécifique. Vous pouvez explorer et copier les extraits qui vous intéressent.

Comment contribuer ?

- Forkez ce dépôt.
- Ajoutez vos extraits de code dans le répertoire approprié ou créez un nouveau répertoire.
- Créez une pull request avec vos modifications.

Merci de documenter vos extraits et de maintenir un format clair et cohérent.
